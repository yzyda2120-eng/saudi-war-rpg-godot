# حرب الصحراء الشاملة (Saudi War RPG)

## أول لعبة RPG حربية سعودية مخصصة للمكفوفين

### الميزات الست الأساسية:

1. **نظام الصوت المحيطي (3D Spatial Audio)**
   - توزيع الصوت في فضاء 360 درجة
   - تحديد اتجاه العدو والرصاص والمركبات
   - "عين" اللاعب المكفوف

2. **نظام القتال والأسلحة (Advanced Arsenal)**
   - أسلحة حقيقية: صقر (رشاش)، شاهين (قناص)
   - محاكاة الارتداد (Haptic Feedback)
   - نظام الذخيرة والإعادة

3. **نظام التصنيع والموارد (Crafting & Vehicles)**
   - جمع الموارد: حديد، وقود، ذخيرة
   - صناعة المركبات: دبابة الفهد، مدرعة النمر
   - تطوير الدروع

4. **نظام التنقل الذكي (AI Navigation)**
   - توجيه صوتي نحو الأهداف
   - حساب المسافات والزوايا
   - تنبيهات العدو القريب

5. **نظام الحماية (Anti-Cheat & Shield)**
   - حماية من الغش
   - نظام الدروع القابل للتطوير
   - التحقق من سلامة البيانات

6. **الواجهة السعودية الذكية (Saudi Accessible UI)**
   - واجهة ناطقة بالكامل باللغة العربية
   - إيماءات سعودية مبتكرة
   - استجابة فورية (Zero Latency)

### البناء والتشغيل:

#### المتطلبات:
- Godot Engine 3.x أو أعلى
- Android SDK (للتصدير كـ APK)
- Kotlin/Java (للتكامل مع Android)

#### التشغيل المحلي:
```bash
godot3 --path /home/ubuntu/SaudiWarRPG_Godot
```

#### التصدير كـ APK:
1. افتح المشروع في Godot
2. انتقل إلى Project → Export
3. أضف Android كمنصة تصدير
4. انقر على Export APK

### الإيماءات السعودية:

- **النقر المزدوج (Double Tap)**: إطلاق نار
- **الضغط المطول (Long Press)**: قراءة الحالة
- **السحب لليمين (Fling Right)**: جمع موارد
- **السحب لليسار (Fling Left)**: صناعة مركبة
